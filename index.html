The GIS Asset Chooser can now be used in one of three modes:

  1. Select Mode Only - Users can select assets from predefined asset layers.
  2. Sketch Mode Only - Users can sketch custom assets on the map. Sketched assets can be points, polylines, or polygons as needed.
  3. Select Mode and Sketch Mode - Users can select assets from predefined asset layers AND sketch custom assets on the map.

Eventually we plan to have a configurable demo app that allows you to try out all three modes in one place with various different layers and assets, but for now we have separate demo pages for each mode.

Use the navigation links above to view demos of each mode.
