GIS Asset Manager UI can be used in one of three modes:

  1. Select Mode - Users can select assets from predefined asset layers.
  2. Sketch Mode - Users can sketch custom assets on the map. Sketched assets can be points, polylines, or polygons as needed.
  3. Select Mode & Sketch Mode - Users can select assets from predefined asset layers AND sketch custom assets on the map.

Use the navigation links above to view demos of each mode.
